# web-chat-client
web-chat的客户端，基于vue3+vite

# 安装依赖&运行
- 需要node16以上版本

安装依赖
```js
pnpm i
```
运行
```js
pnpm dev
```

# 打包构建
```js
pnpm build-only
```
